# g3rb0ise oh-my-zsh theme 

# Installation 

Prerequisites : 
* zsh shell installed
* [oh-my-zsh](https://ohmyz.sh/) installed


Installation procedure : 

```
cd /tmp/
git clone https://github.com/g3rb0ise/g3rb0ise_theme_zsh.git
cd g3rb0ise_theme_zsh
./install

# open a new shell
```
